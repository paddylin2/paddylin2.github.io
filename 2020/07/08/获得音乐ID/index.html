2、网易音乐的 ID 其他获取办法

打开网页版的网易云音乐，找到需要的歌或者歌单，点击生成外链播放器

单曲

歌单

然后我们得到的 HTML 代码中就有ID


如果出现了下图版权问题


我们可以在谷歌浏览器或者Firfox浏览器打开该页面，右击页面任何地方打开元素检查，并找到字样生产外链播放器，如图

他旁边的1404596131就是我们要找的 ID
